## 프로젝트 개요
Delivery Oracle 이란 배달플랫폼으로 웹 사이트로 제작한 배달플랫폼이다.
소비자, 음식점, 라이더, 관리자 기능이 담겨 있고 각 기능은 아래 표와 같다.

## 구현 기능
**유저**
- 사용자 회원가입 : 이메일, 닉네임,비밀번호를 이용하여 로그인
- 로그인 : 로그인 성공시 jwt 토큰 발급
- 사용자의 주문 금액에 따른 등급 부여

   | 등급	|총액|
   | Bronze	| 0 |
  | Silver	| 10000<=X |
  | Gold	| 50000<=X |
   | Platinum	| 100000<=X |
  - 포인트 충전
